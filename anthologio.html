το ανθολόγιο — a gathering place for things that think together.

Quotes I keep returning to, notes I'm working through, fragments that haven't resolved yet. Some are pulled from the main archive. The positions are set by hand. The connections are drawn when something actually touches something else.

how to use it

Hover any entry to see what it connects to. Tap to read in full. The dot on each connection line holds a note on why those two things are linked — hover it to read. Filter by type using the bar at the top. Use ⟿ path to find the route between any two entries.
